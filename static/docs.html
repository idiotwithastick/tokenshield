Setup Guide

Change your base URL. Add one header. Done.

1. Get Your Proxy Key

Sign up on the dashboard to get a free sk-ts-... proxy key. No credit card needed.

2. Change Your Base URL

Point your tool at TokenShield instead of the provider directly.

# Claude (Anthropic) Before: https://api.anthropic.com After: https://your-proxy.repl.co # OpenAI Before: https://api.openai.com After: https://your-proxy.repl.co # Google Gemini Before: https://generativelanguage.googleapis.com After: https://your-proxy.repl.co

3. Add the Proxy Key Header

Add X-Proxy-Key to your requests. Keep your original provider auth header as-is.

# Claude example curl https://your-proxy.repl.co/v1/messages \ -H "x-api-key: sk-ant-your-claude-key" \ -H "X-Proxy-Key: sk-ts-your-proxy-key" \ -H "content-type: application/json" \ -d '{ "model": "claude-sonnet-4-20250514", "max_tokens": 1024, "messages": [{"role": "user", "content": "Hello"}] }' # OpenAI example curl https://your-proxy.repl.co/v1/chat/completions \ -H "Authorization: Bearer sk-your-openai-key" \ -H "X-Proxy-Key: sk-ts-your-proxy-key" \ -H "content-type: application/json" \ -d '{ "model": "gpt-4o", "messages": [{"role": "user", "content": "Hello"}] }'

Proxy Endpoints

Each endpoint mirrors the provider's original API. Request and response bodies are identical.

# Claude (Anthropic) POST /v1/messages # OpenAI POST /v1/chat/completions # Google Gemini POST /v1beta/models/{model}:generateContent

Response Headers

TokenShield adds headers to every response so you know what happened.

# Cache HIT — response came from cache, zero tokens burned X-Cache-Status: HIT X-Tokens-Saved: 1247 X-Proxy-Time-Ms: 2.3 # Cache MISS — forwarded to provider, response cached for next time X-Cache-Status: MISS X-Tokens-Used: 1247 X-Proxy-Time-Ms: 245.1

Account Endpoints

GET /v1/status

Your account stats, cache stats, and savings summary.

# Auth: X-Proxy-Key header curl https://your-proxy.repl.co/v1/status \ -H "X-Proxy-Key: sk-ts-your-proxy-key" # Response { "account": { "tier": "pro", "requests_today": 142, "request_limit": 5000, "total_tokens_saved": 84520, "total_cache_hits": 89, "total_requests": 203 }, "cache": { "entries": 156, "hit_rate": 0.438 } }

GET /v1/savings

Token savings breakdown by provider.

DELETE /v1/cache/clear

Clear your cached responses. Useful if you want fresh responses from the provider.
